Software

The group produces/contributes to several open-source software packages.

g_amide interfaces with the popular molecular dynamics package gromacs (www.gromacs.org) to predict Amide I vibrational frequencies for backbone carbonyl groups in peptides and proteins.

g_spec offers efficient numerical simulation of linear and 2D vibrational spectra. In conjunction with gromacs and g_amide, it offers a complete numerical library for converting protein structures into accurate Amide I vibrational spectra.
